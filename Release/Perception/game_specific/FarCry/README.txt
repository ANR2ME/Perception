This package is to enable the correct FOV in the original Far Cry game.

First, back up the following file:

{game_install_folder}\Far Cry\FCData\Scripts.pak

The copy the Scripts.pak file in this folder into the above location.

In order to trigger the correct FOV, zoom in with your weapon at the start of the level, and the FOV will correct accordingly.

Courtesy of http://pcgamingwiki.com/wiki/Far_Cry#Field_of_view_.28FOV.29
